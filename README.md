## Dependancies
- [Hugo](https://gohugo.io/installation/)

## Development
To develop and preview the site use the blowfish tooling.
Install the CLI tool globally using npm (or other package manager):
```
npm i -g blowfish-tools
```
Run the CLI
```
blowfish-tools
```
